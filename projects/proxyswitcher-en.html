My Projects: Proxy Switcher

ProxySwitcher

Context:

At school, access to Linux connections was frequently blocked, making it difficult to work efficiently. To address this issue, I developed a script to bypass proxy restrictions that rejected connections.

Objective:

The main objective was to simplify the process of switching between my personal network and the school network without having to manually reconfigure the proxy each time. The script was designed to allow entering the proxy's IP address and port during the first connection, while also offering the flexibility to modify these parameters at any time. Additionally, the data needed to be saved in a file for future use.

Skills Developed:

Programming: Developing a script to manage proxy settings.

Network Analysis: Using Wireshark to identify the proxy's IP address and port.

File Management: Saving and retrieving data from a text file via a script.
