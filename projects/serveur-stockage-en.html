My Projects: File Storage and Torrent Management Server

Context:

Storing files can become cumbersome, especially in terms of space and accessibility. External hard drives are often used for backups, but they require physical presence. Additionally, cloud storage solutions are often costly for large amounts of data. To solve this problem, I decided to create my own file storage server.

Objective:

The primary objective was to have storage space accessible from anywhere, without relying on physical media or paid services. To achieve this, I repurposed a defective PC that I repaired and equipped with a 14 TB hard drive. By installing Debian and configuring a static IP address, I made the server accessible via SSH. Additionally, I installed a torrent manager, specifically qBittorrent, to simplify file uploads. The configuration of the web interface for the torrent manager allows easy remote access, secured with a strong password.

Skills Developed:

Hardware Repair and Configuration: Recovering and repairing a defective PC, adding an extra hard drive.

System Administration: Installing and configuring Debian, assigning a static IP address.

Network Management: Opening ports on the router to allow remote access.

Software Usage: Installing and configuring qBittorrent for torrent management.

Cybersecurity: Adding IP rules to allow only known IPs and securing server access with a strong password.
