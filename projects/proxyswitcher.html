Mes projets : Proxy Switcher

ProxySwitcher

Contexte :

Au lycée, l'accès aux connexions Linux était régulièrement bloqué, posant des difficultés pour travailler efficacement. Pour résoudre ce problème, j'ai entrepris de développer un script permettant de contourner les restrictions du proxy qui rejetaient les connexions.

Objectif :

L'objectif principal était de simplifier le processus de basculement entre mon réseau personnel et celui du lycée en évitant de devoir reconfigurer manuellement le proxy à chaque fois. Le script devait permettre de saisir l'adresse IP et le port du proxy lors de la première connexion, tout en offrant la flexibilité de modifier ces paramètres à tout moment. De plus, les données devaient être enregistrées dans un fichier pour une utilisation ultérieure.

Compétences développées :

Programmation : Développement d'un script pour gérer les paramètres de proxy.

Analyse réseau : Utilisation de Wireshark pour identifier l'adresse IP et le port du proxy.

Gestion de fichiers : Enregistrement et récupération des données dans un fichier texte depuis un script.
