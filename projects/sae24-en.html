My Projects: Integrative Project

Context:

As part of the SAE 24 project, the objective is to design and implement the network infrastructure for a small company composed of several departments (finance, HR, technical). This infrastructure must enable secure communication between different departments using VLANs while ensuring internet connectivity for client workstations. A central server will be deployed to manage several critical services such as DHCP, FTP, and a web server.

Objective:

The project involves designing a complete network infrastructure, integrating a switch, VLANs, a router, and a WiFi network, to enable smooth and secure communication between the company's departments. A Windows server equipped with Active Directory will be deployed alongside a Linux server and a telephony server to ensure centralized resource management. Additionally, a web server will be configured to present the infrastructure and document potential security threats, such as DHCP attacks. This server will also provide countermeasures to prevent attacks like DHCP starvation and DHCP spoofing. A dynamic PHP interface will be developed on this server, enabling, among other things, user authentication and script execution to automate certain tasks. The project also includes attack testing and demonstrating countermeasures implemented to secure the infrastructure.

Skills Developed:

Network Configuration: Setting up and managing VLANs, a DHCP server, and IP connectivity.

Network Security: Preventing DHCP attacks (rogue DHCP and starvation).

System Administration: Deploying Windows and Linux servers with critical services (Active Directory, FTP server, etc.).

Web Development: Creating dynamic web pages with PHP for administration and documentation.

Project Management: Task scheduling, teamwork, and managing equipment (switch, router, firewall, etc.).
