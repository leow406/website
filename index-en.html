My projects

ProxySwitcher

Context:

In high school, access to Linux connections was regularly blocked, posing difficulties for efficient work. To solve this problem, I undertook to develop a script to bypass the proxy restrictions that were blocking connections.

Objective:

The main objective was to simplify the process of switching between my personal network and the school's network without having to manually reconfigure the proxy each time. The script should allow entering the IP address and port of the proxy during the first connection, while also providing the flexibility to modify these settings at any time. Additionally, the data should be saved in a file for later use.

Developed Skills:

Programming: Development of a script to manage proxy settings.

Network Analysis: Use of Wireshark to identify the IP address and port of the proxy.

File Management: Saving and retrieving data in a text file.

File Storage and Torrent Management Server

Context:

Storing files can become anoying and complex, especially in terms of space and accessibility. External hard drives are often used for backups, but they require physical presence. Additionally, cloud storage solutions are often paid for large amounts of data. To solve this problem, I set out to create my own file storage server.

Objective:

The main objective was to have storage space accessible from anywhere, without relying on physical media or paid services. To achieve this, I acquired a defective PC that I repaired and equipped with a 2 TB hard drive. By installing Ubuntu and configuring a static IP address, I facilitated access to the server via SSH. Then, I added a torrent manager, specifically qBittorrent, to simplify file uploads. The configuration of the torrent manager's web interface allows easy remote access, with security provided by a strong password.

Developed Skills:

Hardware Repair and Configuration: Recovery and repair of a defective PC, addition of an additional hard drive.

System Administration: Installation and configuration of Ubuntu, assignment of a static IP address.

Network Management: Opening ports on the router to allow remote access.

Software Usage: Installation and configuration of qBittorrent for torrent management.

Computer Security: Setting up IP rules to refuse unknown IP addresses and addition of a strong password to secure access to the server.
