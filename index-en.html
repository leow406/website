My Projects

ProxySwitcher

Context:

In high school, access to Linux connections was regularly blocked, creating difficulties in working efficiently. To solve this issue, I undertook the development of a script to bypass the proxy restrictions that were rejecting connections.

Objective:

The main goal was to simplify the process of switching between my personal network and the school's network without having to manually reconfigure the proxy each time. The script was designed to allow entering the proxy's IP address and port on the first connection, while also offering the flexibility to modify these settings at any time. Moreover, the data was to be saved in a file for future use.

Skills Developed:

Programming: Development of a script to manage proxy settings.

Network Analysis: Use of Wireshark to identify the proxy's IP address and port.

File Management: Storing and retrieving data in a text file from a script.

File Storage Server and Torrent Management

Context:

Storing files can become cumbersome, particularly in terms of space and accessibility. External hard drives are often used for backups but require physical presence. Moreover, cloud storage solutions often charge for large amounts of data. To address this issue, I undertook to create my own file storage server.

Objective:

The main objective was to have a storage space accessible from anywhere, without relying on physical media or paid services. For this, I salvaged a faulty PC, repaired it, and equipped it with a 2 TB hard drive. By installing Ubuntu and setting up a static IP address, I facilitated access to the server via SSH. Next, I added a torrent manager, specifically qBittorrent, to simplify file uploading. Configuring the torrent manager's web interface allows easy remote access, secured by a robust password.

Skills Developed:

Hardware Repair and Configuration: Recovering and repairing a faulty PC, adding an additional hard drive.

System Administration: Installing and configuring Ubuntu, assigning a static IP address.

Network Management: Port forwarding on the router to enable remote access.

Software Usage: Installing and setting up qBittorrent for torrent management.

Cybersecurity: Adding IP rules to allow only known IPs and adding a strong password to secure server access.
