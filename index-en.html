My Projects

ProxySwitcher

Context:

At high school, access to Linux connections was frequently blocked, making it difficult to work effectively. To solve this problem, I undertook the development of a script that would bypass the proxy restrictions that rejected the connections.

Objective:

The main goal was to simplify the process of switching between my personal network and the school's network without having to manually reconfigure the proxy each time. The script was designed to allow entering the IP address and port of the proxy during the first connection, while also offering the flexibility to modify these settings at any time. Additionally, the data was to be saved in a file for future use.

Skills Developed:

Programming: Development of a script to manage proxy settings.

Network Analysis: Use of Wireshark to identify the IP address and port of the proxy.

File Management: Saving and retrieving data in a text file.

File Storage and Torrent Management Server

Context:

Storing files can become cumbersome, especially in terms of space and accessibility. External hard drives are often used for backups, but they require physical presence. Furthermore, cloud storage solutions are typically costly for large amounts of data. To address this issue, I set out to create my own file storage server.

Objective:

The main objective was to have a storage space accessible from anywhere, without relying on a physical medium or paid services. To do this, I salvaged a faulty PC, repaired it, and equipped it with a 2TB hard drive. By installing Ubuntu and configuring a static IP address, I made it easy to access the server via SSH. Next, I added a torrent manager, specifically qBittorrent, to simplify file transfers. Configuring the web interface of the torrent manager allows easy remote access, secured by a robust password.

Skills Developed:

Hardware Repair and Configuration: Salvaging and repairing a faulty PC, adding an extra hard drive.

System Administration: Installing and configuring Ubuntu, assigning a static IP address.

Network Management: Opening ports on the router to enable remote access.

Software Use: Installing and configuring qBittorrent for torrent management.

IT Security: Adding IP rules to allow only known IPs and adding a strong password to secure server access.
